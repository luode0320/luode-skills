# Bug 入口正反例

## 正例

### 正例 1：接口异常

- 现象：调用下单接口返回 500。
- 影响范围：测试环境下所有新订单创建失败。
- 期望：应成功创建订单并返回订单号。
- 实际：返回 500 和空响应体。
- 证据：附带错误日志片段。

### 正例 2：页面偶发异常

- 现象：部分用户打开详情页时白屏。
- 影响范围：仅生产环境、部分老账号。
- 期望：详情页正常展示。
- 实际：页面白屏并出现前端报错。

## 反例

### 反例 1：只有猜测

- “应该是缓存有问题吧。”
- 没有异常现象、范围、环境、期望或实际。
- 结论：不通过。

### 反例 2：只有情绪化描述

- “这个功能又坏了，快修一下。”
- 没有给出问题事实。
- 结论：不通过。
