```md
---

# <标题>

- ...

- ...

1. ...

1. ...

## 权责边界与不负责事项

- ...

## 执行通过 / 驳回标准

- 通过：...
- 驳回：...
```

## 命名建议

- 统一后缀：`-rules`
- 主题前缀按项目约定：例如 `project-` 或业务域前缀
- 保持 ASCII 目录名，便于跨平台工具处理
