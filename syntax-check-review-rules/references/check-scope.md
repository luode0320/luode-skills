# 语法校验范围

## 用途

用于界定这一轮检查到底拦哪些基础问题。

## 应拦下的问题

- 语法错误
- 类型不匹配
- 依赖或模块引用缺失
- 明显的构建入口错误
- 前端 import/export 缺失、重复或路径别名错误
- `.vue` 模板绑定、组件注册、script setup 引用、样式引用中的明显基础错误
- `.tsx`、`.jsx`、`.ts`、`.js` 中 ESLint、TypeScript、vue-tsc、构建命令可稳定发现的基础错误

## 不在本轮主范围

- 未使用导入或变量清理
- 多余空行和纯格式问题
- 命名、注释和实现结构问题
- 功能逻辑是否正确
- 纯格式化排序问题；这类问题归 `cleanup-format-review-rules`
