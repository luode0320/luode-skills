# 说明
改指定需求的需求类别


# url
`${TAPD_API_ENDPOINT}/stories/change_workitem_type`

# 支持格式
JSON/XML（默认JSON格式）

# HTTP请求方式
POST

# 请求数限制
一次只能修改一个需求的类别

# 请求参数
|字段名|必选|类型及范围|说明|特殊规则|
|:----:|:----:|:----:|:----:|:----:|
| story_id |  `是` | integer |  需求ID | |
| workitem_type_id |  `是` | integer |  更改的目标需求类别ID |  |
| workspace_id | `是` | integer | 项目ID |  |
