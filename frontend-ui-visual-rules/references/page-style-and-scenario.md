# 页面风格与场景映射

## 常见页面类型

### SaaS / 后台 / 管理台

- 优先方向：克制、清晰、稳定、信息密度可控
- 推荐关键词：简洁、分层、数据优先、弱装饰、强结构
- 慎用：大面积高饱和渐变、过重玻璃模糊、情绪化字体

### 电商 / 营销 / 落地页

- 优先方向：转化导向、视觉节奏明确、主 CTA 突出
- 推荐关键词：卖点分层、信任感、主行动清晰、模块节奏强
- 慎用：页面所有模块都在争主角、CTA 太多、颜色过杂

### 内容站 / 博客 / 文档

- 优先方向：阅读优先、版面稳定、层级克制
- 推荐关键词：留白、可读性、长文本友好、目录清晰
- 慎用：花哨背景压过正文、正文宽度过宽、正文字号过小

### AI 工具 / 搜索 / Copilot 类

- 优先方向：聚焦输入、结果层级清楚、状态反馈明确
- 推荐关键词：输入主导、对话节奏、结果分层、等待感可控
- 慎用：紫色泛滥、无意义霓虹、输入区和结果区权重失衡

### 移动端消费者产品

- 优先方向：主路径短、触达友好、反馈快速、上下文连续
- 推荐关键词：单手可达、轻操作、节奏快、状态可感知
- 慎用：点击区域过小、依赖 hover、顶部堆太多操作

## 风格选择原则

- 一个页面只允许一个主视觉方向。
- 一个产品线尽量只保留一套主图标语言。
- 暗黑模式不是把亮色反转，而是重新校准层级和对比。
- 先让页面像“这个产品”，再让它像“某种流行风格”。
