# 审美方向与记忆点规则

## 进入视觉设计前先回答 4 个问题

1. 这页是给谁用的
- 是后台操作者、普通消费者、品牌访客，还是内容阅读者。
- 受众不同，信息密度、情绪强度和操作节奏都不同。

2. 这页最重要的目标是什么
- 是转化、浏览、提交、比较、阅读，还是探索。
- 目标不清时，不要先做装饰。

3. 这页的语气是什么
- 尽量选一个明确方向，而不是“什么都沾一点”。
- 可参考：克制极简、编辑感、玩具感、奢华精致、工业理性、复古未来、自然有机、粗野直接、几何装饰、柔和轻快。

4. 这页最值得被记住的一点是什么
- 可能是标题排版、首屏构图、色彩对比、信息节奏、动效开场、插图语言或某个独特容器形态。
- 一页至少应有一个记忆点，但不要让每个模块都抢记忆点。

## 方向确定原则

- 一个页面只保留一个主方向，不要把“科技感 + 杂志感 + 奢华感 + 卡通感”混成一页。
- 极简不等于普通，强表达也不等于堆满效果。
- 先让风格服务场景，再谈“好不好看”。
- 如果项目已有品牌系统，新的表达必须在品牌边界内变化，而不是另起炉灶。

## 表达强度与实现复杂度匹配

- 极简、理性、后台类页面：重点在留白、秩序、对齐、密度和层级，代码不需要为了“高级感”强加过多装饰。
- 展示、品牌、营销、活动类页面：可以增加更强的构图、背景、动效和叙事，但每个效果都要服务主信息。
- 复杂视觉如果无法稳定实现、无法维护、明显拖慢加载，就应主动降阶，而不是硬堆。

## 常见坏味道

- 风格名词很多，但看不出页面到底想传达什么。
- 每个模块都有自己的一套配色、阴影、圆角和排版语言。
- 所谓“高级感”只剩大渐变、玻璃模糊和发光描边。
- 页面没有记忆点，换成任意其他产品也成立。
