# 推理总结结构模板

```text
Skill 命中检查：
- 命中 skill：...

Skill 执行证据：
- ...

当前要解决的问题：
- ...

当前问题的解决方案、问题根因：
- 解决方案：...
- 根因：...

当前解决结果、结论：
- ...

本次改动点：
- ...（仅在本轮有改动时出现）

验证结果：
- ...（仅在本轮有验证时出现）

函数注释核对：
- ...（仅在本轮有代码改动时出现；无函数改动需写“函数位点 0 个”）

下一步建议：
1. ...
2. ...
3. ...
```

## 结构要求

- 固定顺序输出，不要随意重排。
- 每一节至少 1 条可复核信息。
- “下一步建议”必须可执行，且 1-3 条即可。
