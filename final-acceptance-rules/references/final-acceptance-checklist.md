# 最终验收前检查清单

- 是否已经存在对应需求主文档。
- 是否已经存在来源对象（需求或 Bug）对应的验收标准文档。
- 是否已经存在对应实施总览文档；若任务较大，是否也存在对应实施周期文档。
- 实施总览 / 实施周期文档是否已经记录周期顺序、第一期 / 第二期 / 第三期等期次定位和周期内最小任务顺序。
- 本轮验收范围内的全部实施周期是否已经按顺序收口。
- 每个纳入验收范围的最小任务是否已经具备实现、真实测试、审查、验收证据。
- 是否不存在当前周期未收口却提前验收后续周期的情况。
- 是否已经存在正式测试结果。
- 是否已经存在正式审核结果。
- 是否确认需求自上次验收后没有新增会使结论失效的变更。
- 是否已经知道若本轮不通过，应回流到测试、审核、验收标准还是实施文档。
